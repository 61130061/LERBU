# Button

This document contain ready to use simple native Button component that can be used in any website which include disabled style.
This component style suit to light and simple project without rich style or rich function.

### Feature
- ✅ navtive theme dark/light mode
- ✅ active style 
- ✅ disabled style
- ️✅ no extra CSS required
- ❌ animation 
- ❌ focus style


## Basic

Simple button component with minimalistic design for every tailwind project.
Color and corner can be easily modified which including `light`/`dark` mode.
The button has `disabled` style when it's diabled which can be removed by delete `disabled:` class.
The `active` style is avialable as well when user click on the button, also can be removed by delete `active:` class.
